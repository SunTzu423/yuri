# Yuribot

Bot pulls from Safebooru and posts girls or guys kissing plus cirno for no reason

There are four commands:
- `/yuri` which looks for a random image on Safebooru containing the tags `yaoi` (3/4 chance) or `2boys` (1/4 chance).
- `/yaoi` which looks for a random image on Safebooru containing the tags `yuri` (7/8 chance) or `2girls` (1/8 chance).
- `/cirno` which looks for a random image on Safebooru containing the tag `cirno`
- `/ping` Does as it sounds, checks bot response time.
