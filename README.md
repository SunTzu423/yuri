# Yuribot

***WARNING: THIS BOT MIGHT ACCIDENTALLY SEND CONTENT THAT IS QUESTIONABLE.***

A bot that sends pictures of girls kissing (or boys kissing) from Gelbooru.\
It can be added as a bot (as in, inside servers) or as a user application (as
in, you can just use it everywhere).\
There are three commands:
- `/yuri` which looks for a random image on Gelbooru containing the tags `yuri`
  and `kiss`.
- `/yaoi` which looks for a random image on Gelbooru containing the tags `yaoi`
  and `kiss`.
- `/gelbooru` which takes space seperated tags, and finds a random image on
Gelbooru containing those tags. Use with caution!

`/yuri` and `/yaoi` also do not send images containing the `bdsm` tag.
